## Project Description
This project is a part of the Digital Literacy subject. The aim of this project is to learn how different digital tools are used in studies and how to use the internet in a safe and responsible way.

In this project, I completed multiple tasks such as creating an infographic, setting up my profiles on different platforms, solving a basic coding problem, writing formal emails, and studying cybercrime. These tasks helped me understand how digital platforms are useful for students.
